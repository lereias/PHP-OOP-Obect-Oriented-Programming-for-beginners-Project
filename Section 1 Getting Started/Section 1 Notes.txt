DISPLAYING ERRORS IN PHP
========================

htdocs/example/test.php

<?php

phpinfo();

?>

- shows php info
- display_errors should be On if not:
  ~ Open php.ini in xampp/php folder
  ~ find display_errors
  ~ display_errors=On
- output_buffering should be 4096 if not:
  ~ Open php.ini in xampp/php folder
  ~ find output_buffering
  ~ output_buffering=4096
* if there are changes on php.ini file, restart server by stopping apache
  and mysql and starting it again
